# 分散式資源班間接服務紀錄系統 V1.0

專為資源班教師設計的輕量化紀錄工具，旨在簡化間接服務之登錄、統計與報表產出流程。

採用單一 HTML 檔案架構，無需安裝任何軟體或資料庫伺服器即可運作，實現跨平台相容性，資料儲存於本機瀏覽器，不會上傳至任何伺服器。

## 下載與使用

1. 下載 `IndirectServiceTracker_v1.0.html`
2. 以任何現代瀏覽器開啟（Chrome、Edge、Firefox 均可）

詳細操作說明請參閱[使用手冊（V1.0）](https://docs.google.com/document/d/18DdZf7PFNC3p_Gogzd7WKn_BBmuxkNOlxrs0QSDKtq8/edit?usp=sharing)。

## 主要功能

- **紀錄管理**：新增、編輯、刪除間接服務紀錄
- **儀表板**：即時運算總服務時數、服務人次及類型佔比
- **報表輸出**：依日期範圍篩選，列印時自動產生統計圓餅圖
- **CSV 匯出**：可直接以 Excel 開啟
- **資料備份／還原**：JSON 格式匯出與匯入
- **深色模式**

## 已知問題

列印報表時，統計圓餅圖在某些解析度下可能覆蓋表格內容。建議在瀏覽器列印預覽中將縮放比例調整至 90% 或以下。
