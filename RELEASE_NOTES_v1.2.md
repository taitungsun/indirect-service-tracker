# V1.2 — 網站圖示與日常使用體驗修正

本次為維護性更新，新增網站圖示 (favicon)，並修正兩個會直接影響日常紀錄與報表正確性的問題。**建議所有使用者更新**，特別是有早上到校即登錄紀錄習慣的老師。

## 重點更新

### 🆕 網站圖示 (Favicon)
以內嵌 SVG 方式加入系統圖示，瀏覽器分頁與書籤會顯示藍底剪貼板圖案，多分頁工作時更容易辨識。維持單一 HTML 檔、離線可用的特性，不需額外圖檔。

### 🐛 日期預設值時區錯誤（重要）
台灣時區 (UTC+8) 在**每天早上 8 點以前開啟系統時，「日期」欄位會誤預設成「昨天」**，容易在不知情的狀況下誤植紀錄日期。已修正為任何時間開啟都正確顯示今天。

> 📌 若您過去曾在早上 8 點前新增紀錄，建議到「歷史紀錄」頁籤檢查該時段的紀錄日期是否正確，必要時用編輯功能修正。

### 🐛 服務對象統計失真
成果報告中，服務對象為「無 (非諮詢服務)」的紀錄原本會被計入「其他」人次，造成統計虛增。現在「無」獨立成一列，統計總數仍與紀錄筆數一致。

### 🛡️ 穩健性強化
- 編輯紀錄儲存時，若原紀錄已不存在（例如已在另一個分頁刪除），不再靜默遺失修改內容，會提示並改以新增方式保留。
- 「還原 JSON」的檔案選擇器限制為 `.json` 格式，避免誤選其他檔案。

## 升級方式

下載 `IndirectServiceTracker_v1_2.html`，用**同一個瀏覽器**開啟即可——新舊版共用同一份 `localStorage` 資料，不需任何轉移步驟。建議升級前仍先匯出一份 JSON 備份。

完整更新內容請見 [CHANGELOG.md](https://github.com/taitungsun/indirect-service-tracker/blob/master/CHANGELOG.md)。
